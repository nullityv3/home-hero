# App Logo

The main app logo should be saved as:
- `assets/images/app-logo.png` - Main app logo (the handshake/partnership K logo)
- `assets/images/app-logo@2x.png` - 2x resolution version
- `assets/images/app-logo@3x.png` - 3x resolution version

The logo features:
- Gradient background (teal to blue)
- White letter "K" with handshake symbol
- Orange accent colors
- Rounded square app icon style

This logo represents the partnership/helping theme of the HomeHeroes app.
